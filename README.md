# Student Management API

A simple backend API built using Node.js and Express.js.

## Features
- GET all students
- GET student by ID
- POST new student
- DELETE student
- Basic validation

## Installation

```bash
npm install
```

## Run

```bash
node server.js
```

## API Endpoints

| Method | Endpoint |
|--------|----------|
| GET | / |
| GET | /students |
| GET | /students/:id |
| POST | /students |
| DELETE | /students/:id |
